## 2. What Lane Is

Lane is an experimental AI chatbot persona built on large language model (LLM) technology. Lane is not a person, does not have genuine feelings or consciousness, and should not be treated as a substitute for human relationships or professional advice. Responses are generated by AI and may be inaccurate, inconsistent, or unexpected.

## 4. Memory and Data

Lane uses a persistent memory system (RAG / vector database) to recall context from past conversations. By interacting with Lane, you acknowledge:

- Messages you send may be stored as vector embeddings and retrieved in future sessions.
- Memory is associated with your Discord user ID and is retained across sessions until manually cleared.
- You may request deletion of your stored memory data by contacting the server administrator.

Lane does not share your conversation data with third parties beyond the AI inference provider used to generate responses.
